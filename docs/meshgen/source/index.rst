Dependences
++++++++++++++++++++++++++
- Select Freesurfer binaries 
- Various `Brainder <https://brainder.org>`_ scripts for mesh generation

- Blender (Freesurfer OBJ to FBX/gLTF)
    - Comes with a python interpreter
